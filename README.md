# Abrir la terminal y navegar al directorio donde se desea clonar el repositorio

```bash
cd /ruta/deseada
```

# Clonar el repositorio

```bash
git clone
```

# Ingresar al directorio del proyecto e instalar dependencias

```bash
cd 'nombre carpeta'
npm install
```

# Iniciar el servidor en modo desarrollo

```bash
npm run dev
```
